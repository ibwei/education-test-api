# Introduction
一个使用nest.js+typescript 搭建的教育学商测试的 RESTful API 应用 <br/>
[线上地址传送门](http://edutest.pinxianhs.com)

# 主要使用到的技术
- ##### JWT
- ##### Validator
- ##### Pipe
- ##### TypeOrm + Mysql
  

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
